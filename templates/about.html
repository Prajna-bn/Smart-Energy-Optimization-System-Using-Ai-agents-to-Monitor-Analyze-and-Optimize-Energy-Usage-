About Energy Demand Predictor

Project Overview

This application uses machine learning to predict energy demand based on electrical parameters and environmental conditions.

Models Used
Random Forest

Ensemble learning method that constructs multiple decision trees and outputs the mean prediction. Handles non-linear relationships well.

Neural Network

Multi-layer perceptron with multiple hidden layers that can learn complex patterns in the data through backpropagation.

Features
  • Secure user authentication with password hashing
  • Real-time predictions using trained ML models
  • Prediction history tracking for each user
  • Detailed metrics including variance, MSE, and runtime
  • Ensemble predictions combining multiple models
Input Parameters
Parameter Description Typical Range
Voltage (V) Electrical voltage 1 - 13 V
Current (A) Electrical current 0 - 13 A
Temperature (°C) Ambient temperature 25 - 36 °C
Humidity (%) Relative humidity 55 - 80 %
Performance Metrics
  • Convergence Speed: Training time for each model
  • Final Cost: Mean Squared Error (MSE) of predictions
  • Variance: Statistical variance between model predictions
  • Runtime: Time taken to generate prediction
Note: This is a demonstration project. Predictions should be validated with real-world data before use in production.
